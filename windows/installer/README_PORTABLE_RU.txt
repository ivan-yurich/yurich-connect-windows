Aurum VPN Portable

Важно: не запускай AurumVPN.exe прямо внутри ZIP-архива.
Windows в таком режиме открывает только один exe без соседних DLL, поэтому приложение не сможет найти файлы вроде url_launcher_windows_plugin.dll.

Как запустить portable-версию:
1. Нажми "Извлечь все".
2. Открой распакованную папку.
3. Запусти START_AURUM_VPN.cmd или AurumVPN.exe.

Для VPN-туннеля Windows может попросить права администратора. Это нормально: TUN/Wintun требует повышенные права.
