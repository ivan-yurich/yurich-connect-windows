# Решение проблем Yurich Connect

## Не подключается

- Проверьте, выбран ли профиль.
- Нажмите **Обновить пинг**.
- Нажмите **Починить подключение**.
- Попробуйте другой профиль.
- Отправьте диагностический отчёт разработчику.

## Требует права администратора

Это нормально для Windows TUN/Wintun. Нажмите **Перезапустить от имени администратора** и подтвердите UAC.

## После отключения нет интернета

Нажмите **Починить подключение**. Приложение остановит свои процессы, удалит временные конфиги и выполнит `ipconfig /flushdns`.

## Не импортируется подписка

- Проверьте, что ссылка открывается в браузере.
- В Yurich Panel должен быть включён raw/Base64/Xray JSON/sing-box template.
- Попробуйте вставить ссылку из буфера обмена.

## Не работает NaiveProxy

- Проверьте формат `naive+https://`.
- Убедитесь, что `runtime\naive.exe` есть в папке приложения.
- Попробуйте обновить подписку и подключиться заново.

## Не работает VLESS Reality

- Проверьте `uuid`.
- Проверьте `publicKey`.
- Проверьте `serverName/sni`.
- Проверьте `shortId`, если он требуется сервером.

## Windows Defender или SmartScreen ругается

Для публичных сборок желательно подписывать `YurichConnect.exe` и `YurichConnect_Setup.exe`. Если сборка не подписана, SmartScreen может показать предупреждение. Скачивайте файлы только из официального GitHub Releases проекта.

## Как отправить отчёт

Нажмите **Отправить отчёт**. Секреты маскируются автоматически: UUID, пароли, токены, ключи и ссылки подписок.

## Полное удаление

1. Отключите VPN.
2. Закройте Yurich Connect.
3. Удалите приложение через Windows.
4. При необходимости удалите `%APPDATA%\Yurich Connect`.
